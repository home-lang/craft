# Craft Roadmap

This file exists so issue templates and docs link to a real planning surface.

- Stabilize the Zig 0.17 dev toolchain path across local, CI, and release builds.
- Keep the TypeScript SDK, native binary, and scaffolded apps on matching versions.
- Expand package coverage for non-framework integrations before revisiting React, Vue, and Svelte.
- Replace masked CI commands with explicit build, test, and artifact checks.
